## ✨ Features

- 🤖 **Auto-Fetch** - Pulls your real GitHub repos, stars, and languages
- 🖼️ **Auto-Generate** - Creates project screenshots and thumbnails  
- 🎨 **4 Themes** - Ocean, Purple, Forest, Sunset gradients
- 📱 **4 Layouts** - Modern Grid, Minimal List, Masonry, Showcase
- 📸 **Photo Upload** - Use your own profile picture
- 📞 **Contact Cards** - Email, Phone, LinkedIn, Twitter, Location
- ⚡ **Zero Dependencies** - Single HTML file, no build step
- 🎯 **One-Click Export** - Download or copy complete portfolio
